# Component Inventory

## Maturity levels

| Level | Meaning |
| --- | --- |
| Candidate | Potential future component. |
| Planned | Approved direction, not implemented. |
| Experimental | Implemented but API may change. |
| Stable | Production-usable API. |
| Deprecated | Replaced by another pattern. |
