# Commercial Licensing Overview

This page summarizes how VyrnForge UI licensing is intended to be understood by evaluators and future commercial customers. It is informational only and does not replace the root [VyrnForge Source License 1.0](../../LICENSE) or any separate written commercial agreement.

## Evaluation Permission

VyrnForge UI is source-available under the VyrnForge Source License 1.0.

The license permits:

- source inspection;
- local evaluation;
- private non-production evaluation environments;
- temporary internal prototypes used only to evaluate suitability.

These rights do not include production use, commercial use, redistribution, resale, sublicensing, white-labeling, or competing-library use.

## When A Commercial License Is Required

A separate written commercial license is required before using VyrnForge UI for Production Use or Commercial Use.

Examples that require a separate written commercial license include:

- internal administration portals used by employees;
- customer portals or partner-facing applications;
- reporting, analytics, workflow, IAM, access-management, or operations applications used in business operations;
- hosted applications, paid products, or services that incorporate VyrnForge UI;
- distribution of compiled application code that incorporates VyrnForge UI.

Commercial rights are defined by the separate written agreement, not by this overview.

## Application Distribution Versus Resale

A commercial agreement may allow a customer to sell or distribute an application that was built with VyrnForge UI, including compiled application code, when the written agreement says so.

That is different from reselling VyrnForge UI itself. Unless a separate written agreement expressly allows it, you may not:

- redistribute VyrnForge source code or packages;
- republish VyrnForge packages to npm or another registry;
- sell VyrnForge as a standalone library, component kit, template, or development asset;
- sublicense VyrnForge source code;
- white-label VyrnForge as another UI library, component library, design system, or data-grid product;
- use VyrnForge to create, improve, or operate a competing UI, component, design-system, data-grid, UI-builder, low-code, or no-code product.
